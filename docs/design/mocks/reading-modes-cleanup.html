05

Switching modes

Mode is locked per-tale on the server for a good reason (a novel tale is never mid-read asked for choices). Two ways to let readers change their mind — the recommended one ships now with no risky server surgery.

Recommended — Restart to switch

“Read as a Novel” restarts this tale in the other mode. Clean, safe, no server change beyond one parameter — the restart path already carries the mode.

☰ Restart as a Novel
Heavier — Live mid-read switch

Flip a tale’s mode in place; future turns adopt the new shape. More powerful, but reopens the Pro-gating and “prose prompt suddenly asked for choices” edge the current design deliberately closed.

The one decision for you. Everything in 01–04 is client-side and ships regardless. The only fork is how switching works: restart-to-switch (safe, now) vs a live in-place flip (a real server change). I recommend restart-to-switch for this pass and treating the live flip as a follow-up.
